# RandomNumbers
Implementação de um gerador de números Pseudo-Aleatórios, utilizando Congruência Linear, em Python.

- Script criado para atender especificações do projeto [GNUS CASSINO](https://github.com/Gnus-Digitais/GnusCassinosOficial)
- Desenvolvido pela equipe [Gnus-Digitais](https://github.com/Gnus-Digitais)
